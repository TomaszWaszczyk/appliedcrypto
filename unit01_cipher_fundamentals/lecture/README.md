# 1. Fundamentals

## Outline
The key concepts involved in this unit are:

The key concepts are:

* Ciphers.
* Encoding methods (ASCII, UTF-16, Base64, Hex).
* Prime Numbers.
* GCD.
* Large numbers.
* Random Number Generators.
* Data Integrity (CRC-32).
* Frequency Analysis.
* Key sizes.
